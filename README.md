<h1 align="center">Simple Bookmarks</h1>

<p align="center">
  <i>Simple Bookmarks is a modern, free and open source bookmark manager.</i>
</p>

---

<img src="https://simple-bookmarks-495.vercel.app/dashboard-demo-dark.jpg" alt="Simple Bookmarks Logo">

## ✨ Features

- **Collections** - Organize your bookmarks into collections.
- **Tags** - Add tags to your bookmarks with multiple keywords.
- **Rich Preview** - Every bookmark you save is automatically transformed into a rich preview with title and an image.
- **Share collections** - Share your collections with others using a public link.
